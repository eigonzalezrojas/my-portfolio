## 🚀 Instalación y Ejecución Local

1. Clonar el repositorio:
```bash
git clone https://github.com/eigonzalezrojas/my-portfolio.git
cd my-portfolio
```
2. Instalar dependencias:
```bash
npm install
# o
yarn install
```
3. Ejecutar en modo desarrollo:
```bash
npm run dev
# o
yarn dev
```
